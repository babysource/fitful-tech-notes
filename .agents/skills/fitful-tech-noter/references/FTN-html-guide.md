# 网页指导纲领

HTML 呈现页；基于 Markdown 内容源生成可视化复习界面。

## 信息规范

- 依托网页特性，无损重现 Markdown 内容源。
- 术语、代码、命令等专业信息准确呈现。
- 知识脉络通俗易懂、由浅入深、由易到难。
- 内置游戏化记忆辅助与复习测评，聚焦学习主题充分覆盖知识要点。

## 视觉规范

- 按内容规模自适应组织单页或多页。
- 索引目录固定左侧，滚动始终可见。
- 交互体验择优适配，避免装饰过载。
- 禁止横向溢出；内容不得撑破视口。
- 滚动仅作兜底，严禁替代布局治理。
- 以响应式安全约束保障内容完整可见。
- 确保信息层级、导航、可读性与可定位性。
- 紧扣复习目标，巧设轻量直观的延迟反馈游戏。
