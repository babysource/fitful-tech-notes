# 网页指导纲领

HTML 呈现页；基于 Markdown 内容源生成可视化复习界面。

## 信息规范

- 依托网页浏览特性，无损重现 Markdown 内容源。
- 专业信息准确呈现，知识脉络通俗易懂、由浅入深、由易到难。
- 内置延迟反馈游戏辅助记忆与测评，聚焦学习目标充分涵盖知识要点。

## 视觉规范

- 按内容规模自适应组织单页或多页。
- 交互体验择优适配，避免装饰过载。
- 左侧边栏索引、右侧主内容流布局。
- 双栏独立滚动，正文置顶主题横幅。
- 禁止横向溢出；限高容器留足空间。
- 滚动仅作兜底，严禁替代布局治理。
- 以响应式安全约束保障正文内容完整可见。
- 确保信息层级、导航、可读性与可定位性。
- 紧扣复习目标，巧设轻量直观的酷炫游戏。
