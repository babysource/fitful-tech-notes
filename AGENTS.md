# 维护指南

本仓库为静态技术笔记库，`README.md` 是笔记索引入口。

## 索引规范

- 同步更新：新增或调整笔记时，必须同步维护笔记目录与对应大类索引表格。
- 目录结构：笔记路径遵循 `一级目录/二级目录/YYYY-MM-DD/<笔记文件>`。
- 导航目录：以“笔记目录”为入口，使用嵌套列表；一级目录映射大类，二级目录映射类目。
- 图标标识：索引与分类使用 emoji 图标作为视觉标识，且全局唯一，不得复用。
- 分类层级：大类使用二级标题；类目作为索引表格“类目”单元格内容。
- 锚点策略：统一使用中文自定义锚点；大类锚点置于大类标题，类目锚点置于对应类目单元格。
- 索引表格：每个大类维护一个 Markdown 表格，列固定为 `类目 | 笔记 | 日期 | MD | HTML`。
- 内容格式：笔记标题精炼且不含日期；日期统一为 `YYYY-MM-DD`；链接采用 `./` 前缀相对路径，并同时维护 `MD` 与 `HTML`。

## 索引模板

```markdown
# 📚 笔记目录

- [<大类 Emoji> <大类名称>](#<大类锚点>)
  - [<类目 Emoji> <中文类目>](#<类目锚点>)

## <span id="<大类锚点>"><大类 Emoji> <大类名称></span>

| 类目 | 笔记 | 日期 | MD | HTML |
|---|---|---|---|---|
| <span id="<类目锚点>"><类目 Emoji> <中文类目></span> | <精炼笔记标题> | <YYYY-MM-DD> | [MD](./<一级目录>/<二级目录>/<YYYY-MM-DD>/<笔记文件名>.md) | [HTML](./<一级目录>/<二级目录>/<YYYY-MM-DD>/<笔记文件名>.html) |
```
